# Site da Infinita

Projeto em Next.js com páginas separadas, logo da empresa, navegação profissional e integração com WhatsApp.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar na Vercel

1. Envie esta pasta para um repositório no GitHub.
2. Entre na Vercel.
3. Clique em **Add New > Project**.
4. Importe o repositório.
5. A Vercel detectará automaticamente o projeto Next.js.
6. Clique em **Deploy**.

## Observações

- O botão flutuante do WhatsApp e os CTAs já usam o número `55 71 91111-1111`.
- O formulário da página de contato abre o WhatsApp com os dados preenchidos.
- Se depois quiser um chat real no site integrado ao WhatsApp Business API ou DigiSac, essa base pode ser expandida.
